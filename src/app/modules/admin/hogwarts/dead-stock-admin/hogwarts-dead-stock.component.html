Kako radi pricing mode:
`Markup on cost` = uzmi nabavnu cenu iz webshop baze i dodaj maržu iz polja `Value`.
Primer: nabavna 100, value 10, nova cena 110.
`At cost` = prodaja po nabavnoj ceni iz webshop baze.
Primer: nabavna 100, nova cena 100.
`Discount on current price` = uzmi trenutnu webshop cenu i spusti je za procenat iz polja `Value`.
Primer: redovna cena 200, value 20, nova cena 160.
